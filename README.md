# ZHIF - ZH_CN Interactive Fiction (Interpreter)

ZHIF 是一个中文互动小说解释器，基于 Python3 实现。

必要的文件：

- `main.py`
- `zhif.py`
- `config.toml`

## 快速使用

运行 `run.bat` 即可，这将运行默认的 default.zhif。

## 如何编写 zhif 代码

直接写一行文字，将直接输出
在行首写一个 `/`，代表一个指令

### 指令

/color [fore|back|style] [color/style/reset]
/color reset
